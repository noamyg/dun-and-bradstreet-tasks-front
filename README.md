# Task Management Frontend

Angular 21 frontend for the Dan & Bradstreet Task Management API.

## Prerequisites

- **Node.js >= 22.12.0** (required by Angular 21)
  ```bash
  nvm use 22.12.0
  ```
- Backend API running at `http://localhost:39509`

## Installation

```bash
npm install
```

## Development

```bash
ng serve
```

Open `http://localhost:4200/`.

## Build

```bash
ng build
```
